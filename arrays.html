Arrays

In JavaScript, an array is a special type of object that stores a collection of values, which can be of any data type. JavaScript arrays are zero-indexed, meaning that the first element of an array is at index 0, the second element is at index 1, and so on.

To create an array in JavaScript, we can use the array literal syntax, which consists of enclosing a comma-separated list of values within square brackets:

let myArray = [1, 2, 3, "four", true];

We can also create an array using the Array() constructor, which takes one or more arguments that represent the initial elements of the array:

let myArray = new Array(1, 2, 3, "four", true);


1.PUSH - The push() is a built-in method of arrays that adds one or more elements to the end of an array and returns the new length of the array.
2.POP - The pop() is a built-in method of arrays that removes the last element from an array and returns that element. This method changes the length of the array.
3.UNSHIFT - The unshift() is a built-in method of arrays that adds one or more elements to the beginning of an array and returns the new length of the array.
4.SHIFT - The shift() is a built-in method of arrays that removes the first element from an array and returns that element. This method changes the length of the array.
In JavaScript, you can also create a multidimensional array by nesting arrays inside one another. The number of nested arrays determines the number of dimension in the array.
Multidimensional Array - is an array that contains other arrays as its elements, forming a nested structure. Each nested array represents a separate dimension, creating a grid-like or matrix-like structure.
